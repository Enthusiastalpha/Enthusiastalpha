## 📌 Featured Project
### 🏗️ [Azure VMSS — Highly Available Web Infrastructure](https://github.com/Enthusiastalpha/azure-vmss-terraform)
A zone-redundant Virtual Machine Scale Set deployed across 3 Azure Availability Zones with a real-time monitoring dashboard.

**What makes it production-grade:**
- Zero secrets on disk — all credentials live in Azure Key Vault
- Remote Terraform state in Azure Blob Storage
- Real metrics dashboard — CPU, memory, disk, network from `/proc` and Apache logs
- Dynamic NSG rules, variable validation, lifecycle rules, consistent tagging
